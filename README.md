Turf Manager App ⚽

A full-stack portfolio project designed to manage local football/turf groups, streamline roster tracking, and resolve post-match expense splitting.

Core Features:
- Zero-Compliance UPI Ledger: Dynamic UPI deep-link generation for seamless expense splitting without payment gateway overhead.
- Match Engine: Automated 1-tap RSVPs, waitlist management, and a public "Free Agent" toggle to fill empty roster slots.
- AI Database Manager: Integration with the Gemini API to allow natural language querying of the MySQL ledger (e.g., "Who hasn't paid for last week's match?").

Tech Stack:
- Backend: Python (FastAPI), MySQL, SQLAlchemy
- Frontend/Mobile: React, React Native CLI
- AI Integration: Google Gemini API
